# 勤怠データ解析ルール

## 対応フォーマット

### フォーマットA（2026年3月以前）

```
対象月    YYYY-MM
時給（円）    N,NNN
月合計勤務時間    HH:MM
月合計金額（円）    NNN,NNN
日付（曜日）    出勤 (hh:mm)    退勤 (hh:mm)    休憩 (hh:mm)    合計勤務時間
```

- 1日1行、出退勤は1セットのみ
- 合計勤務時間が空の行はスキップ

### フォーマットB（2026年4月以降）

```
対象月    YYYY-MM
案件名    XXX
時給（円）    N,NNN
月合計稼働時間    HH:MM
月合計休憩時間    HH:MM
月合計金額（円）    NNN,NNN
日付（曜日）    稼働1    終了1    稼働2    終了2    稼働3    終了3    日別稼働時間    日別休憩時間
```

- 1日最大3セットの出退勤
- 日別稼働時間が空の行はスキップ
- 案件名フィールドが追加

## 時間変換ルール

HH:MM → 小数時間（小数2桁）:

| 時:分 | 小数 | 計算 |
|-------|------|------|
| 1:08 | 1.13 | 8/60=0.133→0.13 |
| 2:15 | 2.25 | 15/60=0.25 |
| 0:22 | 0.37 | 22/60=0.367→0.37 |
| 7:02 | 7.03 | 2/60=0.033→0.03 |

**端数処理**: 小数第3位を四捨五入

## 請求書への変換

1. 稼働がある日のみ明細行を作成
2. 品目名: `開発業務 M/DD`（月は0埋めなし）
3. 単価: 勤怠シートの時給
4. 数量: 日別稼働時間（小数2桁）
5. 単位: `時間`
6. 消費税: `ten_percent`

## 請求書メタ情報

- タイトル: `YYYY年M月分 業務委託費のご請求`
- 支払期限: 翌月末
- 振込先: ユーザーに確認（初回のみ。以降は過去の請求書から取得）

## ファイル命名規則

PDF提出時: `YYYYMM_氏名_業務委託費用.pdf`
例: `202603_森井祥喜_業務委託費用.pdf`
