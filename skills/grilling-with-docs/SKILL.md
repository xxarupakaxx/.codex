`grilling` を読み、ユーザーが durable artifact の作成に同意した決定だけを記録する。

- 用語の整理には `modeling-domains` を併用する。
- 後戻りしにくい比較判断だけを `creating-adr` の基準で ADR 候補にする。
- 記録先、差分、外部公開は project policy と External Write Gate に従う。
- 合意前の仮説、未決事項、agent の推測を確定事項として保存しない。
