# Evaluator Rubric — 評価ルーブリック

## 4基準の加重スコアリング

### 基準1: デザイン品質（Design Quality）— 重み x2

色・タイポグラフィ・レイアウトが組み合わさって、独自のムードとアイデンティティを創出しているか。

| スコア | 基準 |
|--------|------|
| **1** | デフォルトテンプレート。Bootstrap/Tailwind UIのまま。個性なし |
| **2** | 基本的なカスタマイズ。色は変えたがレイアウトは定型。「よくあるWebサイト」 |
| **3** | 意図のあるデザイン。方向性は明確だが、細部の洗練が足りない |
| **4** | プロフェッショナル品質。Linear/Notion/Stripeクラスのまとまり |
| **5** | 美術館に展示できるレベル。視覚的に記憶に残る独自の世界観 |

### 基準2: 独自性（Originality）— 重み x2

テンプレートやライブラリデフォルトではなく、カスタムな判断の証拠があるか。

| スコア | 基準 |
|--------|------|
| **1** | 完全にテンプレート。AIスロップの兆候（紫グラデーション、汎用ブルー、Inter+白背景） |
| **2** | テンプレートをベースにしたカスタマイズ。構造は定型 |
| **3** | いくつかのカスタム判断。レイアウトまたはインタラクションに独自性 |
| **4** | 明確な独自アプローチ。「このサイトならでは」の要素がある |
| **5** | 予想外の創造的飛躍。新しいインタラクションパターンや空間的体験 |

**AIスロップの検出チェックリスト**:
- [ ] 紫→青グラデーション背景
- [ ] Inter / Arial / Roboto デフォルト
- [ ] 汎用 `#3B82F6` ブルーのCTA
- [ ] teal-coral カラーコンボ
- [ ] Glassmorphism濫用
- [ ] 3カラムカードグリッド（均等配置）
- [ ] "Get Started" / "Learn More" 汎用コピー
- [ ] スプリング・バウンシーアニメーション

1つでも該当すれば独自性は最大3。3つ以上該当で最大2。

### 基準3: クラフト（Craft）— 重み x1

タイポグラフィ階層、スペーシングの一貫性、色の調和、コントラスト比。

| スコア | 基準 |
|--------|------|
| **1** | スペーシング不揃い。フォントサイズがバラバラ。色のコントラスト不足 |
| **2** | 基本的な整列はできている。一部不整合 |
| **3** | 4pxグリッド準拠。タイポグラフィ階層が明確。色の調和あり |
| **4** | ピクセルパーフェクト。一貫したトークンシステム。WCAG AA準拠 |
| **5** | 職人的細部。マイクロインタラクション、カスタムトランジション、完璧なリズム |

### 基準4: 機能性（Functionality）— 重み x1

ユーザーがインターフェースの目的を理解し、主要アクションを発見・実行できるか。

| スコア | 基準 |
|--------|------|
| **1** | 何をするサイトか分からない。ナビゲーション不明 |
| **2** | 目的は分かるが操作に迷う。CTAが不明確 |
| **3** | 基本的なユーザビリティ。主要フローは問題なし |
| **4** | 直感的なUX。情報アーキテクチャが優れている |
| **5** | デライトフルなUX。マイクロインタラクションで操作が楽しい |

---

## 合計スコアの計算

```
合計 = (Design x 2) + (Originality x 2) + Craft + Functionality
最大 = (5 x 2) + (5 x 2) + 5 + 5 = 30 + 5 = 35 ← 修正不要、正しい
```

| 合計 | 判定 |
|------|------|
| 30-35 | 卓越。即座にPASS |
| 25-29 | PASS。十分な品質 |
| 18-24 | ITERATE。改善の余地あり |
| 12-17 | 要大幅改善。PIVOTを検討 |
| <12 | 根本的な方向転換が必要 |

## Few-shot キャリブレーション例

### 例1: スコア 15/35（ITERATE）

```
Design: 2 (4) — カスタムカラーだがレイアウトはHeroセクション+3カラムカードの定型
Originality: 2 (4) — Inter使用、汎用ブルーCTA、"Get Started"コピー
Craft: 3 — スペーシングは概ね4px準拠、タイポ階層あり
Function: 4 — ナビゲーション明確、CTAは見つかる

批評:
- KEEP: スペーシングシステムは良好。維持すべき
- FIX: Inter→コンテキストに合ったフォントに変更。カラーをブランドに合わせる
- FIX: Hero+3カラムのテンプレート構造を脱却。非対称レイアウトやベント式グリッドを検討
- CONSIDER: マイクロインタラクションの追加
```

### 例2: スコア 24/35（ITERATE → PASS近い）

```
Design: 4 (8) — 一貫したダークテーマ、独自のカラーパレット、洗練されたレイアウト
Originality: 3 (6) — カスタムレイアウトだが、インタラクションは標準的
Craft: 4 — ピクセルパーフェクト、トークンシステム一貫
Function: 5 — デライトフルなマイクロインタラクション

批評:
- KEEP: ダークテーマの実装品質。カラーパレットの調和。マイクロインタラクション
- FIX: ナビゲーションのホバー状態をもっと独自にする
- CONSIDER: スクロールに連動した空間的な演出（パララックスではなくCSS perspective等）
```

### 例3: スコア 30/35（PASS — 卓越）

```
Design: 5 (10) — 3Dギャラリー空間として美術館を再構築。チェック柄の床、壁面のアート配置
Originality: 5 (10) — ドアを介した部屋間ナビゲーション。スクロール/クリックではない空間体験
Craft: 4 — CSS perspective描画は精緻。一部のモバイル対応に課題
Function: 5 — 空間的メタファーが直感的に理解でき、探索が楽しい

批評:
- KEEP: 空間的体験のコンセプト全体。CSS perspectiveの使い方。部屋間遷移
- FIX: モバイルビューでのパフォーマンス最適化（will-changeの適用）
- CONSIDER: キーボードナビゲーションの追加（矢印キーで部屋移動）
```

## 出力テンプレート

Evaluatorは以下のフォーマットで出力する:

```markdown
## Evaluation — Iteration {N}

### Screenshots
- Desktop (1440px): {path}
- Tablet (768px): {path}
- Mobile (375px): {path}

### Scores
| Criterion | Raw | Weighted | Notes |
|-----------|-----|----------|-------|
| Design Quality | {1-5} | {x2} | {根拠} |
| Originality | {1-5} | {x2} | {根拠} |
| Craft | {1-5} | {x1} | {根拠} |
| Functionality | {1-5} | {x1} | {根拠} |
| **Total** | | **{/35}** | |

### AI Slop Check
{チェックリスト結果}

### Feedback
#### KEEP（触らないこと）
- {良い部分。具体的に}

#### FIX（必ず修正）
- {問題}: {具体的な修正案。ファイル・行番号・CSS値等}

#### CONSIDER（余裕があれば）
- {改善提案}

### Strategic Recommendation
{REFINE / PIVOT / PASS の推奨と理由}
```
