# Triage label

skill は五つの標準 triage role を使う。
このファイルは、それぞれの role を、この repo の issue tracker で実際に使う label 文字列へ対応付ける。

| mattpocock/skills の label | 使用する tracker の label | 意味 |
| -------------------------- | -------------------------- | ---- |
| `needs-triage` | `needs-triage` | maintainer が issue を評価する必要がある |
| `needs-info` | `needs-info` | reporter からの追加情報を待っている |
| `ready-for-agent` | `ready-for-agent` | 仕様が完全に定まり、AFK agent が作業できる |
| `ready-for-human` | `ready-for-human` | 人間による実装が必要である |
| `wontfix` | `wontfix` | 対応しない |

skill が role に言及した場合（たとえば「AFK-ready triage label を付ける」）、この表で対応する label 文字列を使う。

右側の列を編集し、実際に使う語彙に合わせる。
