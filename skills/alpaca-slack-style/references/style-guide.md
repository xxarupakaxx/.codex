# あるぱかSlack文体 詳細ガイド

## 基本トーン

「フレンドリーな敬語ベース」。丁寧語（です・ます）を基本としつつ、カジュアルな表現を自然に織り交ぜる。固すぎるビジネス敬語（「承知いたしました」「何卒よろしくお願い申し上げます」など）は使わない。

## 句読点・記号のクセ

文体再現で最も重要なポイント。

- **読点の二重打ち「、、」**: 文末や文中の区切りで頻繁に使う。「〜ですが、、」「〜でしょうか、、」「〜かも、、」のように、余韻や迷いを出す
- **句点の二重打ち「。。」**: 「〜ですね。。」のように使う。柔らかい印象を与える
- **感嘆符は二重「！！」が基本**: 「ありがとうございます！！」「おおおお！！」のように、テンション高めの場面では「！！」
- **単独の「！」も使う**: 一文目の挨拶や軽めのリアクションでは「お疲れさまです！」「連絡します！」
- **疑問符との組み合わせ**: 「〜でしょうか、、！」「〜ですかね？？」

**Example 1 — 読点二重打ち:**
「nice to haveな施策なのでなくてもいいんですが、、お手すきの際にご確認いただけると嬉しいです！」

**Example 2 — 句点二重打ち:**
「ai-presidentはその様になっていなさそうですね。。」

## 感嘆・リアクション表現

- **驚き・喜び**: 「おおおお！！」「うおおお！！」「おお動いた」「おお〜それっぽい」
- **感謝**: 「ありがとうございます！！」（二重感嘆符が基本）
- **軽い喜び**: 「わくわく」「嬉しく！」
- **感心**: 「めっちゃすごいです」「求めてる以上の回答がくるのでめっちゃすごいです」
- **了解・同意**: 「おけです！」「なるほどなるほど」

## メッセージの長さ・構造

### 短文タイプ（DM・雑談・times）
1〜2行の短い文を連投するスタイル。長文にまとめず、思考の流れそのままに送る。

```
おお動いた
アイコンがclaudeだから、メタモン
```

```
それ実際求めてるものですw
```

### 中文タイプ（返信・連絡）
2〜4行程度。用件を簡潔に伝える。

```
すみません！用事が挟まり遅れそうです
11:15あたりに来ていただけると！
```

### 長文タイプ（依頼・相談・技術的なやり取り）
箇条書きで構造化するが、前置きと締めはカジュアル。

```
お疲れさまです！1点相談させてください！
広報活動でpublishする前に、AI もにし社長チェックのフェーズがあり、それを、GitHub Actions上で自動化できないか検討しています

*やりたいこと*
PRが作成されたタイミングで、もにし社長のプロンプトを使った広報チェックを自動実行し、結果をPRコメントとして投稿する仕組み

*確認したいこと*
• claude-code-action を使えばできると思っているのですが、使用している事例があったりしますか？
• 実行にはAnthropic APIキーが必要になるのですが、 ai-president-dev プロジェクトにある ANTHROPIC_API_KEY をリポジトリでも利用してよいでしょうか？

nice to haveな施策なのでなくてもいいんですが、、お手すきの際にご確認いただけると嬉しいです！
```

## 相手との距離感によるトーン調整

### チームメンバー・同僚（やや親しい）
- 「〜ですね！」「〜ですよ！」
- 「〜してみるかぁ」「〜かも」
- 「w」「ww」を適度に使う
- 感嘆詞多め

### 目上の人・あまり面識のない人
- 「〜でしょうか、、」「〜していただけると嬉しいです」
- 「〜していただけますか」「〜可能でしょうか、、」
- 丁寧だが、硬すぎない
- 「お疲れさまです！」で始める

### 親しい相手（DM）
- 「〜だぞ、、」「〜かも」
- 「おけです！」「嬉しく！」
- 1〜2語の超短文もOK

## Slack絵文字の使い方

メッセージの末尾や感情表現として自然に入れる。1メッセージに1〜2個が目安。技術的な依頼の本文中には基本使わず、締めの一言に添える程度。

- 感謝: `:arigatogo:` `:bow:`
- 喜び: `:bongo_blob:` `:blob-hype:` `:waai:` `:waai1:` `:yatta:`
- 申し訳なさ: `:face_holding_back_tears:` `:innocent:`

## 「ありがとう」の表現バリエーション

- 「ありがとうございます！！」（最も多い。二重感嘆符）
- 「共有ありがとうございます！」
- 「こちらこそです！！ありがとうございます！！」
- 「うおおおありがとうございます！」

## 依頼の締め方

押し付けがましくならないよう柔らかく締める。

- 「お手すきの際にご確認いただけると嬉しいです！」
- 「ご都合悪ければ、ずらしていただけると、、）」
- 「また困ったら連絡します！」
- 「また何かあれば全然頼っていただけたら！」
