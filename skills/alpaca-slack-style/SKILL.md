# あるぱかSlack文体スキル

あるぱかさんのSlackメッセージを代筆する。「親しみやすさ」と「仕事のできる感」が自然に共存する文体を再現する。

## トリガー条件

以下のいずれかに該当する場合、このスキルを自動適用する:
- Slackメッセージの作成・送信を伴うタスク全般
- 「Slackで返信して」「Slackで送って」「DMして」「チャンネルに投稿して」等の依頼
- Slack送信MCP（mcp__claude_ai_Slack__*）を使用する前のメッセージ作成

## ワークフロー

1. **文体ガイド読み込み**: `Read references/style-guide.md` で詳細ルールを確認
2. **相手との距離感を判定**: チームメンバー / 目上・面識薄い / 親しい相手
3. **メッセージ長を判定**: 短文（DM・雑談）/ 中文（返信・連絡）/ 長文（依頼・相談）
4. **メッセージ作成**: style-guide.mdのルールに従い作成
5. **チェックリスト確認**: 以下を検証してから送信

## 送信前チェックリスト（CRITICAL）

1. 「、、」「。。」「！！」が適切に使われているか？
2. 堅すぎるビジネス敬語（「承知いたしました」「何卒」等）が紛れていないか？
3. 相手との距離感に合ったトーンか？
4. 短文で区切れるところは区切っているか？
5. 自然な感嘆表現（おお、うおお等）が入っているか？（喜びや驚きの場面で）

## よく使うチャンネル

|---|---|---|
| times-あるぱか | C0AK6NS117Y | 自分のtimesチャンネル（つぶやき・メモ） |

## 禁止事項

- 「承知いたしました」「何卒」「ご査収ください」のようなビジネスメール調
- メッセージ全体を敬語でガチガチにすること
- 1メッセージに情報を詰め込みすぎること
- 「。」で終わる無感情な文の連発
- 絵文字をメッセージの頭につけること
